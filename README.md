## 功能
1. 查詢北科學生課表
2. 查詢北科當學年行事曆
3. 查詢北科 i 學園公告
4. 下載北科 i 學園檔案
5. 查詢北科學生成績與學分資訊
6. 支援中文、英文顯示
7. 消息推播

## 環境安裝問題
### 缺少 google-services.json
可以到 [TAT 的雲端](https://drive.google.com/drive/folders/1uGzZfcNeYGytfZp8z7QPqSftCaQH1jfK?usp=sharing)上下載 Firebase 相關檔案，並根據裡面的文件進行設定。
